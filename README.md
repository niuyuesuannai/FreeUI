# 使用 React+typescript 从零到一打造一套组件库
### 安装
~~~javascript
npm install yefreeui --save
~~~
### 使用
~~~javascript
// 加载样式
import 'yefreeui/dist/index.css'
// 引入组件
import { Button } from 'yefreeui'
~~~
###  相关技术
* 🔥typescript with React Hooks
* ⛑️使用 react-testing-library 完成单元测试
* 📚使用 storybook 本地调试和生成文档页面
